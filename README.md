### 蒼穹のファフナー とは

平和な日常が突如奪われ、痛みや喪失を抱えながら成長する群像劇が描かれる。

2004年に第1期が放送、2023年に完結。\

### このプロジェクトのストーリー

<div align="center"><h5>ソロモンに応答あり！！</h5></div>

フェストゥムの襲来を検知。第一種警戒態勢によりヴァッフェラーデン スタンバイ。偽装鏡面解除、第一・第二ヴェルシールドが展開されるが、
攻撃により、主電源とCDC統合制御基盤の接続が断絶。補助電源に移行するが、長くは持たない。
そこで、真壁 一騎が駆るマークアレスが機体の同化能力を介したエネルギーを直接供給する。
パスワードが合えば、ブリュンヒルデ・システムが起動できる。

| 要素         | 設定                                       |
| ------------ | ------------------------------------------ |
| PC           | ブリュンヒルデ・システム                   |
| CPU          | ジークフリード・システムの演算コア         |
| GPU          | CDCの映像制御兼ウルドの泉 補助演算ユニット |
| メモリ       | ウルドの泉 バッファメモリ                  |
| ストレージ   | アルヴィスの地図データ含む全データ         |
| 電源ユニット | 主電源                                     |
| マザボ       | CDC統合制御基盤                            |
